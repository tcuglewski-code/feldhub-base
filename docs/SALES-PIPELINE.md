## Pipeline Stages

```
LEAD → QUALIFIED → DEMO → PROPOSAL → WON
                                   ↘ LOST
```

### Stage 1: LEAD 🔍
*Erstkontakt, Interesse vorhanden*
- Quelle: Website, LinkedIn, Empfehlung, Kaltakquise
- Aufgabe: Branche, Firmengröße, Kontakt dokumentieren
- Next Step: BANT-Check → Qualifizierung

### Stage 2: QUALIFIED ✅
*Budget, Authority, Need, Timeline geprüft*
- Budget: >500€/Monat vorstellbar
- Authority: Entscheider-Kontakt hergestellt
- Need: Konkreter Bedarf formuliert
- Timeline: Entscheidung in <6 Monaten
- Next Step: Demo-Termin vereinbaren

### Stage 3: DEMO 🎥
*Live-Demo durchgeführt*
- Demo-URL: https://demo.appfabrik.de
- Demo-Script: 15 Min Walkthrough
- Zugänge: admin@demo.appfabrik.de / Demo2026!
- Next Step: Individuelles Angebot erstellen

### Stage 4: PROPOSAL 📄
*Angebot versendet*
- Setup-Fee + monatliche SaaS-Gebühr
- AVV + AGB beilegen
- Entscheidungsfrist setzen (14 Tage)
- Next Step: Follow-Up nach 3 Tagen

### Stage 5: WON 🎉 / LOST ❌
*WON:* Vertrag unterschrieben → Onboarding starten  
*LOST:* Grund dokumentieren → Lessons Learned
