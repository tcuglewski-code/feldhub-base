# FELDWERK AppFabrik — Produktportfolio

> Stand: 29.03.2026 | Version 1.0

## Executive Summary

FELDWERK AppFabrik bietet drei Produktpakete für Field-Service-Management:
- **Basic** — Einzelunternehmer & kleine Teams (1–5 Nutzer)
- **Pro** — Mittelständische Betriebe (6–25 Nutzer)
- **Enterprise** — Große Unternehmen & Konzernstrukturen (25+ Nutzer)

Jedes Paket ist White-Label-fähig und vollständig an die Kundenbranche anpassbar.

## 🎯 Branchen-Konfigurationen

Jedes Paket kann branchenspezifisch konfiguriert werden. Die Konfiguration erfolgt über `tenant.ts`.

### Forstwirtschaft (Referenz: Koch Aufforstung)

**Aktivierte Branchenmodule:**
- ✅ Förderberatung (GAK, Landesförderprogramme)
- ✅ Abnahmeprotokolle
- ✅ Qualifikationsnachweise (Motorsägen-Schein etc.)
- ✅ Saatguternte-Tracking
- ✅ Flächenmanagement (GeoJSON)

**Angepasste Labels:**
- Auftrag → "Pflanzauftrag"
- Mitarbeiter → "Pflanzer"
- Gruppen → "Pflanzkolonnen"
- Protokoll → "Pflanzprotokoll"

**Spezielle Auftragstypen:**
- Aufforstung, Flächenvorbereitung, Kulturschutz, Kulturpflege
- Saatguternte, Zaunbau, Jungbestandspflege, Beratung

**Empfohlenes Paket:** Enterprise (wegen Förderberatung + Offline-Modus)

## 🔧 Technische Basis

Alle Pakete nutzen dieselbe technische Plattform:

- **Frontend:** Next.js 14 + Tailwind CSS
- **Backend:** Prisma + PostgreSQL (Neon)
- **Mobile App:** React Native + Expo (WatermelonDB für Offline)
- **Hosting:** Vercel (99.9% Uptime)
- **Auth:** NextAuth + 2FA (TOTP)
- **Dateien:** Nextcloud-Integration

**Deployment:**
- 1 Vercel-Projekt pro Tenant
- 1 Neon-Datenbank pro Tenant
- Konfiguration über `tenant.ts`
